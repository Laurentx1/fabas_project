KRAKEN ADMINISTRATIVE SYSTEM

O Sistema de Gestão Administrativa kraken é um conjunto de módulos multiusuários integrados para o planejamento de recursos da empresa e gerenciamento da cadeia de suprimentos. É completo, inovador e capaz de desenvolver todas as atividades administrativas de qualquer tipo de empresa, idealizado e desenvolvido para gestão de empresas que trabalham organizadas, com vistas na certificação de qualidade. Com movimentação totalmente descentralizada, os lançamentos realizados pelos usuários alimentam a base de dados integrada, evitando o trabalho repetido de entrada de informações e garantindo a consistência dos dados. A interface do sistema com o usuário é padronizada, ou seja, todas as formas de interação são apresentadas nos mesmos modelos. O kraken permite também que unidades com conexão on-line possam utilizar o sistema de forma remota, trabalhando diretamente nabase de dados corporativa.
